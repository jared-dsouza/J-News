# Northcoders News

A social news aggregation, web content rating, and discussion website built with React.

## About

Northcoders News allows users to:

- Browse articles by topic
- Read and vote on articles
- Comment on articles
- Upvote and downvote comments

## Tech Stack

- **Frontend:** React, Vite
- **Backend API:** Node.js, Express, PostgreSQL
- **Styling:** CSS (or your chosen styling solution)
